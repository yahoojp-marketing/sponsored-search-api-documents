# DownloadBulkJobStatus (enum)
Status of bulk downloading.
### Service
+ [BulkService](../services/BulkService.md)

| Value | Description | 
|---|---|
| IN_PROGRESS| Job is in progress. |
| COMPLETED| Job is completed. |
| TIMEOUT| Job is time out |
| SYSTEM_ERROR| System error. |
