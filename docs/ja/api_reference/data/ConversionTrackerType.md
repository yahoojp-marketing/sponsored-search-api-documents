# ConversionTrackerType (enum)
ConversionTrackerTypeは、コンバージョンの種別を表します。
### Service
+ [ConversionTrackerService](../services/ConversionTrackerService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| WEB_CONVERSION| string| ウェブページのコンバージョンです。 |
| APP_CONVERSION| string| アプリのコンバージョンです。 |
