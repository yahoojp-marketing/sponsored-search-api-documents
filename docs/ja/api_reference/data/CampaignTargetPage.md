# CampaignTargetPage
CampaignTargetPageオブジェクトは、取得されるキャンペーンのターゲティング設定に関するエントリーを表します。
### Service
+ [CampaignTargetService](../services/CampaignTargetService.md)

| フィールド | データ型 | 説明 | 
|---|---|---|
| Page(inherited)|||
| totalNumEntries| xsd:int| 取得される項目の総件数です。 |
| Page.Type| xsd:string| このインスタンスのPageのサブタイプです。 |
| CampaignTargetPage|||
| values[]| <a href="./CampaignTargetValues.md">CampaignTargetValues</a>| 処理の結果を含む、キャンペーンのターゲティング設定です。 |
