# MarkupLanguageType (enum)
MarkupLanguageTypeは、サイトのマークアップ言語を表します。
### Service
+ [AdGroupAdService](../services/AdGroupAdService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| HTML| string| HTML言語です。 |
| CHTML| string| CHTML言語です。 |
| XHTML| string| XHTML言語です。 |
| WML| string| WML言語です。 |
| ALL| string| 全てを対象とします。 |
