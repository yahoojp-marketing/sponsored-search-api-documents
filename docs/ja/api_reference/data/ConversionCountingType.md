
# ConversionCountingType (enum)
ConversionCountingTypeは、コンバージョンの計測方法を表します。
 
### Service
+ [ConversionTrackerService](../services/ConversionTrackerService.md)

| Enumeration | Type | Description |
|---|---|---|
| ONE_PER_CLICK | xsd:string | ユニークコンバージョンです。 |
| MANY_PER_CLICK | xsd:string | 総コンバージョンです。 |
