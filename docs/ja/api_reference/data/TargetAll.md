# TargetAll (enum)
TargetAllは、全ユーザを配信対象設定です。

### Service
+ [AdGroupRetargetingListService](./services/AdGroupRetargetingListService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| ACTIVE| string| ターゲットに対する入札単価調整のみです。 |
| DEACTIVE| string| ターゲットに対する入札単価調整と掲載先の絞り込みです。 |
