# ReportLanguage (enum)
ReportLanguageは、レポート定義の言語選択を表します。

### Service
+ [ReportDefinitionService](../services/ReportDefinitionService.md)

| Enumeration | Type | Description | 
|---|---|---|
| JA| string| 言語：日本語です。|
| EN| string| 言語：英語です。|
