# ReportDownloadFormat (enum)
ReportDownloadFormatは、定義されたダウンロードレポートのファイル形式を表します。
### Service
+ [ReportDefinitionService](../services/ReportDefinitionService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| XML| string| XMLフォーマットです。 |
| CSV| string| CSVフォーマットです。<br>カンマで区分されます。 |
