# LogicalRuleOperand
LogicalRuleOperandは、組合せ対象のターゲットリストを保持するオブジェクトです。

### Service
+ [RetargetingListService](../services/RetargetingListService.md)

| フィールド | データ型 | max<br>Occurs | min<br>Occurs | resp<br>onse | add | set | remove | 説明 | 
|---|---|---|---|---|---|---|---|---|
| targetListId| long| 1| 1| ○| Req| Req| -| ターゲットリストIDです。<br>組合せのターゲットリストは設定できません。|
