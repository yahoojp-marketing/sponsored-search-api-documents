# ReportDownloadUrlValues

| error[]| <a href="./Error.md">Error</a>| エラーの内容です。 |
| ReportDownloadUrlValues|||
| reportDownloadUrl| <a href="./ReportDownloadUrl.md">ReportDownloadUrl</a>| レポートの情報を含む、ダウンロードURLです。 |
