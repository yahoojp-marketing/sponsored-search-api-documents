# AccountType (enum)
AccountTypeは、アカウントの料金支払い種別を示します。
### Service
+ [AccountService](../services/AccountService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| PREPAY| string| クレジットカードや銀行振込などの前払い方式です。 |
| INVOICE| string| 代理店クライアントなどの請求書方式です。 |
