# EnhancedCpcEnabled (enum)
EnhancedCpcEnabledは、コンバージョン数の最大化設定フラグを指定します。

### Service
+ [CampaignService](../../services/CampaignService.md)

### Namespace
[CampaignService#Namespace](../../services/CampaignService.md#namespace)

| Enumeration | Type | Description |
|---|---|---|
| FALSE | xsd:string | 入札設定からコンバージョンの最大化を解除する |
| TRUE | xsd:string | 入札設定にコンバージョンの最大化を設定する |
