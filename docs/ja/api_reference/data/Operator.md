# Operator (enum)
Operatorは、処理を表す演算子です。
### Service
add, set, remove操作共通

| 値 | データ型 | 説明 | 
|---|---|---|
| ADD| string| 新規追加です。 |
| REMOVE| string| 削除です。 |
| SET| string| 更新です。 |
